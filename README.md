# Tuf_second_round


A high-fidelity, premium fintech dashboard built with React and Vanilla CSS. This project features a modern **Bento-grid** layout, vibrant pastel aesthetics, and editorial-grade design elements like handwritten annotations and floating status badges.

##  Features

- **Asymmetric Bento Grid**: A responsive, multi-column layout using modern CSS Grid techniques.
- **Micro-Animations & Transitions**: Subtle rotations (like the left-tilted activation card) and hover effects for a "premium" feel.
- **Editorial Elements**: 
  - **Handwritten Annotations**: Custom 'Caveat' font integration with loop-de-loop SVG arrows.
  - **Floating Status Badges**: Context-aware badges (Last seen, Activated) with connective visual lines.
- **Data Sections**:
  - **Company Profile & Activation**: Real-time progress bars and engagement metrics.
  - **Celebrate Milestones**: Milestone alerts formatted with a professional "Report" aesthetic.
  - **Automatic Enrichment**: AI-powered profile enrichment with circular map status overlays.
  - **CRM Integration**: Seamless visual representation of Salesforce and HubSpot data syncs.

##  Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Styling**: Vanilla CSS (Global tokens, Grid, Flexbox)
- **Typography**: Plus Jakarta Sans (UI), Caveat (Handwriting)
- **Icons**: Custom SVGs & High-quality PNG assets

##  Installation

1. **Clone the repository**:
   ```bash
   cd frontend_task
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

##  Design Philosophy

This dashboard avoids the generic "flat" UI trend by incorporating **organic elements** (handwriting, tilted cards, overlapping layers) to make the data feel alive and human. The pastel color palette is carefully balanced to ensure professional readability while remaining visually striking.
